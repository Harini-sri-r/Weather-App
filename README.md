# Weather-App
A simple and responsive Weather Application built using HTML, CSS, and JavaScript. It fetches real-time weather data using the OpenWeatherMap API and displays current weather details for any city entered by the user.

-- Features

- Search weather by city name
- Displays current temperature in Celsius
- Shows humidity percentage
- Displays wind speed
- Weather icons change based on conditions (Rain, Clouds, Clear, Mist, etc.)
- Error message for invalid city names
- Responsive and clean UI

-- Technologies Used
 - HTML5 – Structure
 - CSS3 – Styling & layout
 - JavaScript (ES6) – Logic & API handling
 - OpenWeatherMap API – Real-time weather data
